# Jarvis Frontend - React App

This is the React frontend for the Jarvis AI Assistant.

## Development

```bash
npm install
npm start
```

The app will run on http://localhost:3000

## Build

```bash
npm run build
```

## Components

- **App.jsx** - Main application component
- **ChatMessage.jsx** - Message bubble component
- **ChatInput.jsx** - Input field with send button
- **StatusIndicator.jsx** - Connection status display
